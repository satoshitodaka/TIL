# 各種設定
## config/environment/development.rb
### redis-rails
- チュートリアルで指示がある下記の記述は `development.rb` のみ書くこと。　production.rb はlocalhostを使用しないため。
```
config.cache_store = :redis_store, "redis://localhost:6379/0/cache", { expires_in: 90.minutes }
```
